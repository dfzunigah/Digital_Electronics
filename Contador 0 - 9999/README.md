##Universidad Nacional de Colombia
###Electrónica Digital I 
###2010-I

*Proyecto desarrollado para la materia Electrónica Digital I de la Universidad Nacional de 
Colombia primer semestre de 2010*

Este proyecto fue desarrollado en el marco de la materia Electrónica Digital I bajo la dirección
del profesor Sebastían Eslava (jseslavag@unal.edu.co), consiste en un contador de 0 a 9999 con 
reset y pulsador para conteo manual programado en el lenguaje Verilog para la tarjeta Nexys™2 Spartan-3E FPGA Board

Este repositorio lo comparto con el fin de servir de guía para todos aquellos que están haciendo
desarrollos similares y puedan verse atascados en diferentes puntos.

En este momento no estoy haciendo este tipo de proyectos  por lo que no seguiré desarrollando 
el código, si alguien está interesado puede descargar el código y usarlo de la manera mas 
conveniente.
